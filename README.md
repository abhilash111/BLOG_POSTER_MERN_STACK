## Documentation

### Code Documentation

The project uses [JSDoc](https://usejsdoc.org/) for documenting code. All doumentation is available in the docs/index.html

- Install JSDoc globally

```sh
sudo npm install -g jsdoc
```

- Generate/Update Docs

```sh
jsdoc src -r -d docs
```
